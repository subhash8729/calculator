This calculator was one of my **initial projects** when I started learning **web development**.

- CSS styling
- JavaScript DOM manipulation
- Event handling

- Clean and simple UI
- Interactive button click handling

## Technologies Used

The purpose of building this project was to strengthen my basics in frontend development and improve problem-solving skills while learning JavaScript.
